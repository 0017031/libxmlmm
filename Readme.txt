
libxmlmm

1. About

Libxmlmm is a C++ wrapper for libxml 2.0 that only relies on standard C++ and 
libxml2.


2. Install

To install the library you use the traditional "GNU tipple jump":

    ./configure
    make
    make install


3. Copying
